# WP Login Timeout

A lightweight WordPress plugin that allows administrators to set custom user login session timeouts directly from the dashboard.

---

## 📖 Overview

**WP Login Timeout** provides administrators with complete control over WordPress user session durations.

By default, WordPress manages login sessions using predefined expiration periods. This plugin allows you to customize how long users remain logged in, helping you balance convenience and security according to your website's requirements.

Whether you're running a membership site, corporate portal, educational platform, or multi-user application, WP Login Timeout helps reduce security risks associated with inactive user sessions.

---

## ⚙️ How It Works

1. Install and activate the plugin.
2. Navigate to the plugin settings page.
3. Specify the desired login session timeout duration.
4. Save your settings.
5. Users will automatically be logged out when their configured session expires.

## 🚀 Installation

1. Download the plugin ZIP file.
2. Navigate to **WordPress Admin → Plugins → Add New**.
3. Click **Upload Plugin**.
4. Select the ZIP file and install.
5. Activate the plugin.
6. Configure your preferred login timeout settings.
