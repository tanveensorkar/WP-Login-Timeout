# Wp-Login-Timeout

A lightweight WordPress plugin designed to improve security, user session management, and account access control.

## Requirements

- WordPress 6.0+
- PHP 7.4+
- Administrator access

## Installation

1. Download the plugin ZIP file.
2. Go to WordPress Admin → Plugins → Add New.
3. Click Upload Plugin.
4. Select the ZIP file and install.
5. Activate the plugin.
6. Configure settings from the WordPress dashboard.

Short Description

A lightweight WordPress plugin that allows administrators to set custom user login session timeouts directly from the dashboard.

Full Description

WP Login Timeout gives site administrators complete control over how long users remain logged into a WordPress website. Instead of relying on WordPress's default session duration, you can define a custom timeout period that better matches your site's security requirements.

Features
Set custom login session timeout from the admin panel
Improve website security by limiting inactive sessions
Lightweight and easy to configure
No coding required
Works with standard WordPress authentication
Use Cases
Membership websites
Corporate portals
Educational platforms
Multi-user WordPress installations
