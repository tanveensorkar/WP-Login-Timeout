# Wp-Login-Timeout

A lightweight WordPress plugin designed to improve security, user session management, and account access control.

## Requirements

- WordPress 6.0+
- PHP 7.4+
- Administrator access

## Installation

1. Download the plugin ZIP file.
2. Go to WordPress Admin → Plugins → Add New.
3. Click Upload Plugin.
4. Select the ZIP file and install.
5. Activate the plugin.
6. Configure settings from the WordPress dashboard.
